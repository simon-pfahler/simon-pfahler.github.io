Detailed Description

template<class T, pRC::Size D>
class cMHN::L2Regulator< T, D >

Class storing an L2 Regulator, specializes the Regulator class.

This regulator penalises off diagonal elements of the theta matrix with their L2-norm. The member function lambda() can be used to get and set the regularization strength.

Only the regularization strength of this regulator can be changed.

Ttype used for calculations
