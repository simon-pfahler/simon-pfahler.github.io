Detailed Description

template<Size... Ns>
class pRC::Tensor< Bool, Ns... >

Class storing tensors with boolean entries.

This class is a specialization of the general Tensor class for Tensor's with boolean entries.

See Tensorish Types in pRC for more details.

Returns a Tensor of the Tensor class with a single non-zero entry.

The class of the value passed has to be compatible with the Tensor's value class, meaning the Tensor's value class (Bool) has to be constructible from an object of class X. Additionally, the number of indices passed has to match the Tensor's dimension.
