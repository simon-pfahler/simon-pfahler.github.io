Detailed Description

template<class T>
struct pRC::NumericLimits< T, If< IsValue< T > > >

Class storing numeric limits of pRC value types.

This class has only static member functions, and the functions mirror functions from std::numeric_limits. The only addition is tolerance().

Template Parameters
TpRC value class whose numeric limits the class should represent
