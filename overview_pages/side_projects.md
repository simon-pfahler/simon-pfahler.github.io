I have had a few side projects over the years.\
The most important of these is an ongoing effort to make colorblind-safe typesetting easier in LaTeX.
This started a few years ago with the creation of the [colorblind LaTeX package](https://ctan.org/pkg/colorblind) which provides colorblind-safe color palettes.\
Recently, I started helping in the creation of the [cvd](https://github.com/jolars/cvd) LaTeX package, a package that tries to tackle CVD-simulation directly in LaTeX.

This list gives an overview over the more polished of my projects.

- [Simulating color-vision deficiency](https://github.com/jolars/cvd)
- [Easy colorblind-safe typesetting in LaTeX: the colorblind package](https://github.com/simon-pfahler/colorblind)
- [Rendering and exploration of Mandelbrot and Julia sets](https://github.com/simon-pfahler/fractals)
- [EQL&div; network implementation for neural symbolic regression, written in TensorFlow2](https://github.com/simon-pfahler/EQL_div)
